# 八字命盘助手设计规范

## 设计目标

现代中文工具站。页面要清楚、可信、温润，有克制的东方细节，适合手机阅读和微信截图；不做寺庙风、算命广告风或欧美 SaaS 模板。

## 颜色

- 朱砂红 `#9B3A31`：主操作、重点状态。
- 墨色 `#26231F`：标题、深色结果封面。
- 米纸白 `#F4F0E7`：页面背景。
- 纸白 `#FFFDF8`：表单与结果面板。
- 暖金 `#B58A4A`：分隔和东方细节，不用于大面积文字。
- 松石绿 `#456B5B`：安全、完成和辅助状态。
- 暖灰 `#6E675E`：正文说明。

## 字体

- 页面标题：宋体体系，优先 `Songti SC`、`STSong`、`SimSun`。
- 正文与表单：系统中文黑体，优先 `PingFang SC`、`Microsoft YaHei`。
- 数据：使用等宽数字特性，不混用英文展示命理概念。

## 组件

- 面板圆角统一为 `6px`，边框为暖灰细线，阴影只做轻微层次。
- 主按钮使用朱砂红实色，按钮高度不低于 `44px`。
- 二元选择使用分段控件，布尔设置使用复选框。
- 命盘表格允许手机横向滚动，不压缩文字。
- 结果封面使用墨色底和命盘圆环，方便单独截图分享。

## 间距

- 页面内容最大宽度 `1280px`。
- 主要页面区块垂直间距为 `56—80px`。
- 面板内边距为 `20—24px`。
- 表单字段间距为 `16—20px`。
