# 玄枢设计规范

## 设计目标

品牌首屏采用“暗色天文仪器”方向，把东方命理、结构化数据和年轻科技感结合起来；进入工具后回到清楚、可信、温润的中文工具界面。整体不做寺庙风、算命广告风或通用欧美 SaaS 模板。

## 颜色

- 黑曜石 `#04090D`：品牌首屏主背景。
- 朱砂红 `#AA4135`：首屏主操作、重点状态。
- 象牙白 `#EAE3D7`：暗色首屏主标题。
- 古铜金 `#C7A46A`：星仪、品牌图标和数据刻度。
- 矿物青 `#6F9495`：数据关系和辅助状态。
- 墨色 `#26231F`：工具页标题、深色结果封面。
- 米纸白 `#F4F0E7`：页面背景。
- 纸白 `#FFFDF8`：表单与结果面板。
- 暖金 `#B58A4A`：工具页分隔和东方细节，不用于大面积文字。
- 松石绿 `#456B5B`：安全、完成和辅助状态。
- 暖灰 `#6E675E`：正文说明。

## 字体

- 页面标题：宋体体系，优先 `Songti SC`、`STSong`、`SimSun`。
- 正文与表单：系统中文黑体，优先 `PingFang SC`、`Microsoft YaHei`。
- 数据：使用等宽数字特性，不混用英文展示命理概念。

## 组件

- 面板圆角统一为 `6px`，边框为暖灰细线，阴影只做轻微层次。
- 主按钮使用朱砂红实色，按钮高度不低于 `44px`。
- 二元选择使用分段控件，布尔设置使用复选框。
- 命盘表格允许手机横向滚动，不压缩文字。
- 结果封面使用墨色底和命盘圆环，方便单独截图分享。
- 品牌首屏使用真实星仪视觉素材，网页文字与按钮不得烘焙进图片。
- 首屏支持轻微指针视差；系统开启“减少动态效果”时自动关闭。

## 间距

- 品牌首屏内容最大宽度 `1328px`，工具页最大宽度 `1280px`。
- 主要页面区块垂直间距为 `56—80px`。
- 面板内边距为 `20—24px`。
- 表单字段间距为 `16—20px`。
