# 玄枢设计规范

## 设计目标

品牌首屏采用“暗色天文仪器”方向，把传统文化、结构化数据和年轻科技感结合起来；公开首页与登录后工具使用同一套清楚、可信的中文产品界面。整体不做寺庙风、算命广告风或通用欧美 SaaS 模板。

## 颜色

- 基础背景 `#0F1114`：全站主背景。
- 抬升背景 `#15181D`：表单、报告和功能面板。
- 朱砂红 `#C25344`：主要操作和重点状态。
- 主文字 `#F3EDE3`：标题与关键信息。
- 次文字 `#CBBFB0`：正文与说明。
- 古铜金 `#D8B48E`：仪轨、章节和数据刻度。
- 矿物青 `#5D7C92`：数据关系和辅助状态。
- 木 `#6F8F65`、火 `#C25344`、土 `#9D7A56`、金 `#B9B1A6`、水 `#5D7C92`。

## 字体

- 页面标题：宋体体系，优先 `Songti SC`、`STSong`、`SimSun`。
- 正文与表单：系统中文黑体，优先 `PingFang SC`、`Microsoft YaHei`。
- 数据：使用等宽数字特性，不混用英文展示命理概念。

## 组件

- 功能面板圆角使用 `10—16px`，边框为低透明度铜金细线，阴影只用于报告和表单层级。
- 主按钮使用朱砂红实色，按钮高度不低于 `44px`。
- 二元选择使用分段控件，布尔设置使用复选框。
- 命盘表格允许手机横向滚动，不压缩文字。
- 结果区使用章节目录和连续报告结构，重点卡片方便单独截图分享。
- 品牌首屏使用真实星仪视觉素材，网页文字与按钮不得烘焙进图片。
- 首屏报告预览使用缓慢浮动，章节滚动显现；系统开启“减少动态效果”时自动关闭。

## 间距

- 品牌首屏内容最大宽度 `1328px`，工具页最大宽度 `1280px`。
- 主要页面区块垂直间距为 `74—104px`。
- 面板内边距为 `20—24px`。
- 表单字段间距为 `16—20px`。
