# 中文文案规范

1. 使用简体中文和全角标点，说清楚用户能做什么。
2. 标题简短直接，不使用“极致、颠覆、赋能、开启命运之旅”等空泛表达。
3. 按钮使用动词，例如“立即排盘、重新填写、复制结果、生成分享文案”。
4. 表单使用中文用户熟悉的名称：出生日期、出生时间、出生地、日历类型、是否闰月。
5. 错误提示说明具体问题和改法，不单独显示 `Error`。
6. 不在用户界面使用 Four Pillars、Ten Gods、Luck Cycle 等英文命理名词。
7. 不恐吓用户，不做绝对判断，不声称可以改变或精准预测命运。
8. 分析内容保持“仅供传统文化研究、娱乐和自我探索参考”的边界。
9. 中文与英文、数字之间适当留空，库名称只出现在技术文档中。
10. 对隐私做真实描述：浏览器记忆与账号数据分开说明，不承诺不存在的本地存储方式。
