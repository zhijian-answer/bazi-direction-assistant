# UI Rules

## Visual Language

- 东方体系感、轻科技感、克制的高级感。
- 主色为深黑、铜金、暗红，五行使用低饱和自然色。
- 主题图只作为氛围资产，不能把带有文字和按钮的整张设计图当网页背景。
- 功能卡圆角不超过 16 px，不使用卡片套卡片。

## Layout

- 首页顺序固定为 Hero、价值、流程、报告章节、排盘表单、结果预览、FAQ、页脚。
- 桌面端首屏左右布局；移动端先内容后预览。
- 页面固定格式元素必须有稳定尺寸，避免加载和交互导致布局位移。

## Interaction

- 主按钮高度至少 48 px，移动端触控区至少 44 px。
- 图标按钮使用 Lucide 图标并提供可访问名称。
- 二元选择使用分段控件，多项选择使用原生选择器或菜单。
- 所有焦点态清晰可见。

## Motion

- 动效用于建立层级，不用于装饰堆叠。
- 入场 0.5-0.7 秒，按钮反馈 0.15-0.2 秒，环境浮动 8-12 秒。
- `prefers-reduced-motion` 下取消位移、循环和滚动动画。
